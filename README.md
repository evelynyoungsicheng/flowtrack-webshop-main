# FlowTrack Webshop

A fully accessible webshop for productivity tools and accessories, built with WCAG 2.2 AA compliance in mind.

## Features

- **WCAG 2.2 AA Compliant**: Full accessibility support including keyboard navigation, screen reader compatibility, proper color contrast, and touch target sizes
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Consistent Visual Design**: Unified color scheme, typography, and layout throughout
- **Accessibility Statement**: Comprehensive documentation of accessibility features and commitments
- **Copyright & Licenses Documentation**: Complete attribution for all media used

## Pages

- **Home** (`index.html`) - Main landing page with featured products and features
- **Products** (`products.html`) - Full product catalog with filtering options
- **About** (`about.html`) - Company mission, values, and team information
- **Contact** (`contact.html`) - Contact form and business information
- **Accessibility** (`accessibility.html`) - Detailed accessibility statement
- **Copyright & Licenses** (`copyright.html`) - Media attributions and license information

## Accessibility Features

### Perceivable
- Text alternatives for all non-text content
- Color contrast ratios meeting WCAG AA standards (4.5:1 minimum)
- Resizable text up to 200%
- Semantic HTML structure

### Operable
- Full keyboard accessibility
- Skip navigation links
- Visible focus indicators
- Touch targets minimum 44×44 pixels
- Reduced motion support

### Understandable
- Language attributes on HTML
- Consistent navigation
- Clear form labels and error messages
- Predictable page structure

### Robust
- Valid, semantic HTML5
- ARIA landmarks and labels
- Compatible with assistive technologies
